# QuickBite Delivery Intelligence

This package contains a complete presentation deck built around a delivery analytics project.

## Files
- `QuickBite_Delivery_Intelligence.pptx` - presentation deck
- `QuickBite_Delivery_Intelligence.pdf` - PDF export of the deck
- `QuickBite_Delivery_Intelligence.ipynb` - notebook used to reproduce the analysis
- `data/orders.csv` - synthetic order dataset
- `data/quickbite.db` - SQLite database for SQL queries
- `assets/` - charts, map preview, dashboard preview, and tables

## Project idea
An urban food-delivery analytics workflow that covers:
- data collection and wrangling
- EDA and visualization
- SQL analysis
- Folium map preview
- Plotly dashboard prototype
- late-delivery classification

## Model result
The classification model reaches about:
- Accuracy: 0.885
- ROC-AUC: 0.962
